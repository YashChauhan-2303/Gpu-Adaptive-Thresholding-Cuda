🚀 GPU-Accelerated Adaptive Thresholding using CUDA

CUDA implementation of adaptive thresholding for document binarization. This project demonstrates how GPU parallelism can dramatically accelerate image-processing workloads.

Up to 193× faster than the CPU implementation while producing identical output.

View Source Code

📌 Project Overview

Adaptive thresholding converts grayscale images into binary images by computing a local threshold for every pixel. Unlike global thresholding, it performs well under uneven lighting, shadows, and faded text.

🔍 Key Takeaways

  • One CUDA thread processes one pixel.
  • Highly parallel workload with minimal thread dependency.
  • GPU acceleration becomes increasingly effective as image size grows.
  • CPU and GPU outputs were verified for correctness.
  • Achieved up to 193× speedup compared to sequential execution.
